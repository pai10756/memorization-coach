# 二、設計意圖

這是給台灣小學五年級學生使用的白話文課文學習工具。

這篇是白話議論文，不適合一開始就要求孩子逐字硬背七段課文。

工具的教育目標是幫助孩子建立：

1. 文章骨架記憶：知道七段各在講什麼，以及為什麼這樣安排。
2. 關鍵資訊提取：記住重要數據、年份、案例與關鍵詞。
3. 因果鏈理解：能說出塑膠垃圾如何從海洋漂流、分解、進入食物鏈，最後影響人類。
4. 結構化複述能力：看著提示，用自己的話說出整篇文章。
5. 原文還原能力：先理解白話大意，再逐步貼近課文原文。
6. 議論文閱讀力：認出「提出疑問 → 案例揭密 → 說明危害 → 量化嚴重 → 解決方案 → 呼籲行動」的文章結構。

如果遇到 UI、互動或內容呈現需要自行判斷的地方，請優先選擇「能促進孩子主動回想、分段理解、圖像記憶、逐步還原原文」的方案，而不是只追求視覺好看。

核心學習原理：

- Dual Coding：每一段都要有 GPT Image 2 生成的多格漫畫圖，讓文字和圖像互相幫助記憶。
- Chunking：每段壓縮成記憶標籤與背誦骨架。
- Retrieval Practice：用遮蔽、填空、排序、問答促進主動回想。
- Spaced Practice：分段學習，不一次塞滿。
- Elaboration：把抽象環保議題連結到孩子日常生活。

每張圖片都要：

- 橫向 16:9。
- 適合放入 HTML 的學習卡片。
- 多格漫畫 / 教學資訊圖風格。
- 使用水彩風、海洋教育插畫、現代課本輔助教材感。
- 色彩柔和，主色為海洋藍、沙灘米、珊瑚橘、淡綠色。
- 適合台灣小學五年級學生。
- 不要寫實照片風。
- 不要恐怖、血腥、過度悲傷。
- 每格畫面清楚，能幫助背誦順序。
- 文字盡量少；如果要放字，請只放短標題與關鍵提示。
- 所有段落圖片風格必須一致。

圖片處理方式：

1. 請先用 GPT Image 2 依照七段 imagePrompt 生成七張多格漫畫圖。
2. 將圖片以 base64 內嵌到 HTML 中，讓 HTML 可以單檔離線使用。
3. 如果圖片 base64 導致檔案過大，請仍保留圖片欄位與 `<img>` 結構，並在 HTML 中註明可替換圖片的位置，但優先目標仍是 base64 內嵌。
4. 不要依賴外部圖片 URL。
5. 不要使用外部 CDN。
6. 圖片 alt 文字要描述該段學習重點。
